# Game_Penambah_IQ
Game Penambah IQ adalah platform Brain Training interaktif yang dirancang untuk mengasah kognitif, logika, dan memori jangka pendek. Lewat berbagai mini-game strategis seperti Random Number Stream dan tantangan logika real-time, user diajak melampaui batas kecepatan berpikir. Tersedia mode Solo dan Mabar untuk kompetisi asah otak yang intens!
